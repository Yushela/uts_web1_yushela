Kosentrasi Computer and Network Security

Cyber security berasal dari dua kata dalam bahasa Inggris yakni cyber yang artinya dunia maya (internet) dan security yang artinya keamanan. Secara istilah, cyber security merupakan bentuk perlindungan terhadap sistem yang terhubung ke internet. Ini termasuk perangkat keras, perangkat lunak hingga data yang kamu miliki.

Security Engineer

Bertugas membangun dan memelihara sistem keamanan pada database perusahaan.

Security Architect

Memiliki tugas untuk mengawasi sistem keamanan yang sudah dibangun security engineer.

Security Analyst

Posisi ini memiliki tanggung jawab untuk mendeteksi lalu mencegah ancaman dari cyber crime.

Developer Software Security

Bertanggung jawab untuk mengembangkan perangkat lunak keamanan yang tersambung ke sistem.

Auditor Security

Bertugas untuk memeriksa sistem keamanan yang sedang dibangun dan mengeceknya apakah sudah cukup kuat atau belum.

Chief Information Security Officer

Bertanggung jawab sebagai kepala tim cyber security yang akan mengatur sistem keamanan sistem informasi dari seluruh anggota tim.
